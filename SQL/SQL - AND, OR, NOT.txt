
  ## SQL AND => Muestra los registros cuando la condicion separada por AND sea TRUE
  
  $ SELECT [column1, column2] FROM [table_name] WHERE [condition1] AND [condition2];
  

  ## SQL OR =>  Muestra los registros cuando la condicion separada por OR sea TRUE
  $ SELECT [column1, column2] FROM [table_name] WHERE [condition1] OR [condition2];


  ## SQL NOT => Muestra los registros excluyendo la condicion
  $ SELECT [column1, column2] FROM [table_name] WHERE NOT [condition1];
