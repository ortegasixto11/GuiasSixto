Vamos a calcular la distancia entre dos puntos, para esto debemos aplicar la operación matemática para hacer esto.

Podemos definir los puntos como objetos, para esto los escribimos como
const p1 = {    x:0,    y:4  }    const p2 = {    x:3,    y:4  }    

Luego podemos crear una función que reciba dos objetos, calcule la distancia y retorne la diferencia
function distancia(p1,p2) {    const x = p1.x - p2.x    const y = p1.y - p2.y      return Math.sqrt(x  x + y  y)  }    distancia(p1, p2)
