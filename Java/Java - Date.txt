
  ## Date
  
  # Initialization
  Date date = new Date();
  
  # Formatting
  SimpleDateFormat ft = new SimpleDateFormat("");
  ft.format(date);
  
  # Codes
  y => 2001
  M => July or 07
  d => 12 (Day in month)
  h => 12 (Hour format 12)
  H => 20 (Hour format 24)
  m => 30 (Minute)
  s => 55 (Second)
  S => 234 (Millisecond)
  E => Tuesday (Day in Week)
  D => 360 (Day in Year)
  w => 40 (Week in Year)
  a => PM (AM/PM)
